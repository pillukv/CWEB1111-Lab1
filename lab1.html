2018 NBA All-Star Rosters: A Visual Comparison

This chart plots each players scoring volume (Points Per Game) and scoring efficiency (True Shooting rate)







Win Shares (WS), Box Plus Minus (BPM), and Value Over Replacement Player (VORP) are three statistics that attempt to quantify a player's contribution to a team beyond the basic statistics like Points and Rebounds that show up in a box score. This chart shows each team's average for each statistic.
